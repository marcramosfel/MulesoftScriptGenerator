csv_to_type_logical.process_csv_file(csv_file_path, output_directory)¶

Processa o arquivo CSV fornecido e gera um arquivo DataWeave com base nos dados.

Parâmetros:

  • csv_file_path (str) – Caminho do arquivo CSV de entrada.

  • output_directory (str) – Diretório onde o arquivo DataWeave será salvo.

Levanta:

  • FileNotFoundError – Se o arquivo CSV não for encontrado.

  • KeyError – Se um tipo de dado não for encontrado na lista de dbtypes.

csv_to_type_logical.process_csv_files(input_directory, output_directory)¶

Processa todos os arquivos CSV em um diretório fornecido.

Parâmetros:

  • input_directory (str) – Diretório contendo arquivos CSV.

  • output_directory (str) – Diretório onde os arquivos DataWeave serão salvos.
